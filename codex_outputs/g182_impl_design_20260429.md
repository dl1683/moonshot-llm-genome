**GPT-2 Embed Anchor**
Do not match token IDs. Token IDs are meaningless across vocabularies.

Build a frozen Qwen donor embedding target in recipient vocabulary:

```python
@dataclass
class AnchorTargets:
    input_target: torch.Tensor       # [recipient_vocab, 768]
    lm_head_target: torch.Tensor     # [recipient_vocab, 768]
    row_mask: torch.BoolTensor       # rows with valid text mapping
    metadata: dict[str, Any]
```

Pipeline:

1. Load Qwen3 donor via `g165.load_trained_donor()`.
2. Extract donor input embedding and lm head.
3. If donor hidden size is not 768, fit a frozen donor-only PCA projection to 768 dimensions. No training labels, no recipient data.
4. For Qwen3-arch, same tokenizer: target row `i = projected_qwen_row[i]`.
5. For GPT-2-arch:
   - decode GPT-2 token `rid` using `gpt2_tok.decode([rid], clean_up_tokenization_spaces=False)`;
   - re-encode that string with Qwen tokenizer using `add_special_tokens=False`;
   - if Qwen IDs exist, target row is the mean of projected Qwen donor rows;
   - skip GPT-2 EOS/PAD and empty/unencodable strings.
6. Store coverage stats and abort if coverage is unexpectedly low, e.g. `< 0.95`.

Anchor gradient injection mirrors g181a/g165:

```python
def inject_anchor_grad(model, targets, lambda_anchor):
    emb = model.get_input_embeddings().weight
    mask = targets.row_mask.to(emb.device)
    tgt = targets.input_target.to(emb.device, emb.dtype)
    emb.grad[mask].add_(emb.detach()[mask] - tgt[mask], alpha=2.0 * lambda_anchor)
```

For tied embeddings, anchor the shared parameter once. If output head is untied, also anchor `lm_head.weight`.

Choose `lambda_anchor` by initial gradient-L2 matching, not a blind constant:

```python
lambda_anchor = target_anchor_grad_l2 / (2.0 * sqrt(masked_frobenius_sq))
```

The target anchor force should be computed per seed from the Qwen3-arch embed-anchor reference, then reused for GPT-2 so GPT-2 is not advantaged or punished by vocab size.

**LOAO CV**
Two folds:

```python
for heldout_arch in ["qwen3_arch", "gpt2_arch"]:
    train = [r for r in rows if r["arch"] != heldout_arch]
    test = [r for r in rows if r["arch"] == heldout_arch]
```

For every model/baseline:

1. Select alpha inside `train` only using `GroupKFold` over seeds.
2. Fit scaler/imputer on `train` only.
3. Predict `test`.
4. Report MSE, R2, AUROC, kill/continue.
5. Block-bootstrap by seed on the held-out architecture, preserving all arms per sampled seed.

PASS is evaluated per fold, not just pooled.

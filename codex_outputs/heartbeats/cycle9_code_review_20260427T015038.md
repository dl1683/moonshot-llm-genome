- `Sev8` [genome_161_run.py](</C:/Users/devan/OneDrive/Desktop/Projects/AI Moonshots/moonshot-llm-genome/code/genome_161_run.py:255>) silently leaks eval data into LR selection on the fallback path. Exact change: in the fallback, load `N_C4_EVAL + 256` file-backed validation examples too; then `raise RuntimeError` if `len(sel_val_pool) < N_C4_EVAL + 256`, and always slice `sel_texts = sel_val_pool[N_C4_EVAL:N_C4_EVAL + 256]`.

- `Sev7` [genome_160_transport_guided_student.py](</C:/Users/devan/OneDrive/Desktop/Projects/AI Moonshots/moonshot-llm-genome/code/genome_160_transport_guided_student.py:361>) can silently reuse the wrong KD cache.

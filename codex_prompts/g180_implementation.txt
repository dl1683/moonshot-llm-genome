## Design (cycle 66 spec + advisor refinement)

**Concept:** at ≤3% training, extract tokenizer+embed geometry features and use them to predict final run quality (paired final C4 NLL gain vs scratch). Beat early-loss-only baseline by ≥25% MSE on held-out cells.

**Features (8 picks per cycle 66):**
1. Mid-layer spectral invariant: centered hidden-state SVD, tail slope α + participation ratio + sqrt(PR)·α
2. Depth spectral drift: slope of #1 across early/mid/late blocks
3. TwoNN intrinsic dimension on pooled hidden states
4. kNN-10 clustering coefficient (existing primitive)
5. **PCA-64 Procrustes/RSA distance to trained Qwen3 reference** (★ load-bearing per advisor — captures tokenizer/embed alignment)
6. Gradient-noise scale (variance/mean² over 4 microbatches)
7. Curvature proxy: top eigenvalue from 4-6 HVPs on loss (last-block + lm_head if full HVP exceeds 30s)
8. Norm/variance depth ratios

**Label:** paired final C4 NLL gain vs same-seed scratch baseline = `final_c4_nll(scratch) - final_c4_nll(arm)`.

**Architecture span:** train on Qwen-family completed cells (g165/g167/g172/g174/g177/g177v2), test on g173 Llama-family cells. RWKV/Falcon-H1 only for feature-stability sanity, no labels.

**Pass:** at ≤3% training compute, `early_loss + geometry` reduces held-out-family MSE by ≥25% versus `early_loss_only`, with paired bootstrap improvement above 0. Secondary actionability guard: must NOT recommend stop on any arm whose true final gain ≥+0.5 nats.

**Envelope:** 2.0-2.8h hard cap 4h. Most labels from existing JSONs; for missing early checkpoints, replay only first ≤3% of exact cell with same seed/arm.

## Required script structure

1. `extract_features(model, probe_batch, layer_indices) -> dict[str, float]` — computes all 8 features
2. `replay_early_checkpoint(arm_spec, target_steps) -> model_state` — loads + reruns first ≤3% if not in cache
3. `compute_label(arm, seed, results_jsons) -> float` — extracts paired final NLL gain
4. `fit_baseline(features, labels) -> sklearn.linear_model.Ridge` — early_loss only baseline
5. `fit_full(features, labels) -> sklearn.linear_model.Ridge` — all features
6. `evaluate_held_out(model, X_test, y_test) -> dict[str, float]` — MSE, R², bootstrap CI
7. `build_summary(...) -> dict` with PASS/INTERMEDIATE/FAIL gating per pass criteria
8. `main()` orchestrates feature extraction across all known cells, fits both models, evaluates, writes JSON

## Design (envelope-fit)

**Arms (4):**
1. `scratch_ce` — no anchor, plain CE
2. `full_anchor` — λ=0.01 anchor on ALL Qwen3 weights (reproduces g165)
3. `embed_lm_head_only_anchor` — anchor ONLY on `model.embed_tokens.weight` and `model.lm_head.weight` (and tied weights if applicable). λ tuned so initial ‖∇L_anchor‖ matches arm 2's at step 0.
4. `no_embed_lm_head_anchor` — anchor on ALL weights EXCEPT embed_tokens + lm_head. λ tuned same.

**Per-arm λ normalization:**
- Compute Frobenius² of donor − init for arm 2 (full = full Frobenius²).
- For arm 3: F²_embed = sum of squared diffs over embed/lm_head only.
- For arm 4: F²_no_embed = full − F²_embed.
- λ_3 = λ_full × √(F²_full / F²_embed)
- λ_4 = λ_full × √(F²_full / F²_no_embed)

This makes initial anchor-gradient L2 norm equal across arms 2/3/4. Save λ values + Frobenius² breakdown in result JSON.
